Welcome to the CE C/C++ Toolchain!ÔÉÅ

The CE C/C++ Toolchain allows you to create incredible programs for the TI-84 Plus CE/TI-83 Premium CE (with or without Python) using the standard C/C++ language.

If this is your first introduction to the CE Toolchain, check out the Getting Started page. You can join us on the CE Programming Discord server in the #ez80-dev channel by clicking the below invite button if you have any questions along the way.

The toolchain works natively across all major OSes, and has been developed by a few people who enjoy the challenge. If you are interested in contributing your expertise, check out the Contributing to the CE Toolchain page and visit the GitHub repository.
