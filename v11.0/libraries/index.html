CE Library Headers

The CE Libraries are used to aid in rapid development by implementing optimized routines for graphics, fonts, file I/O, keypad input, USB access and more. The libraries are delivered as AppVars that reside in the calculator’s archive, and are dynamically linked at runtime by the LibLoad library. This helps to reduce the size of most programs, and ensures that programs can take advantage of library bug fixes and improvements without needing to be modified or recompiled.

The libraries can be downloaded here: tiny.cc/clibs.

Common libraries:

  • graphx.h
  • fontlibc.h
  • fileioc.h
  • keypadc.h
  • libload.h

USB Libraries:
